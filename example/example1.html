jQuery.confine Example 1

This example demonstrates explicitly confining a textarea via options passed to the jQuery.confine plugin.

Steps to test:

  1. Try typing in more than 50 characters, note the triggering of the 'maxlength' event
  2. Once the maximum is met, note that other keystrokes are blocked, not just trimmed.
  3. While still at the maximum, note that modifiers and control keys can still be used (ctrl-x, ctrl-v, delete, etc)
  4. While still at the maximum, place the cursor in the middle of the block of text and note that typing does not truncate the tail end of the text
